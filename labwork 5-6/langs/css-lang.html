Краткое описание

CSS, или Cascading Style Sheets, является языком стилей, используемым для описания внешнего вида документа, написанного на языке разметки, таком как HTML. Он был разработан W3C в 1996 году и с тех пор постоянно развивается и обновляется.

  • CSS используется для определения стилей элементов веб-страницы, включая размер шрифта, цвет текста, расстояние между параграфами и другие аспекты дизайна. Это позволяет веб-разработчикам контролировать и управлять внешним видом веб-страницы.
  • CSS позволяет разделить содержимое и дизайн веб-страницы, что упрощает обслуживание и обновление сайта. С помощью CSS можно изменить внешний вид всего сайта, изменив только один файл стилей.
  • CSS используется вместе с HTML и JavaScript для создания полноценных веб-страниц. HTML предоставляет структуру веб-страницы, CSS определяет ее внешний вид, а JavaScript добавляет интерактивность и динамику.

CSS разрабатывается и поддерживается консорциумом W3C, который определяет стандарты для веба. Это гарантирует, что CSS будет работать надежно и эффективно на различных платформах и устройствах.

Пример исходного кода 

body {
    background-color: lightblue;
}

h1 {
    color: navy;
    margin-left: 20px;
}

p {
    color: red;
}
